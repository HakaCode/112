<p>European Response Package (ERP) is a community modpack for the game, <a title="Steam Store" href="https://store.steampowered.com/app/793460/112_Operator/?curator_clanid=33067827" target="_blank" rel="noopener noreferrer">112 Operator</a>. Much like <a href="911/otl">On the Line</a>, our previous mod for <a title="Steam Store" href="https://store.steampowered.com/app/503560/911_Operator/" target="_blank" rel="noopener noreferrer">911 Operator</a>, ERP aims to increase the quantity of in-game content (calls, incidents, vehicles etc.) while also providing support for all base languages and extending support for further workshop languages. Call voiceovers will be available only in English, but for the list of supported text languages, please see the <a href="#languages">languages</a> tab below.</p>

<p><img src="images/flags/complete.png" alt="complete" />&nbsp;&nbsp;Item is complete and functional in English. Translations may not be completed.<br /><img src="images/flags/incomplete.png" alt="incomplete" />&nbsp;&nbsp;Item is a work-in-progress and not yet functional in-game.<br /><img src="images/flags/911.png" alt="911" />&nbsp;&nbsp;Item is based on another asset from the 911 Operator Workshop or base game.<br /><img src="images/flags/unique.png" alt="unique" />&nbsp;&nbsp;Item is unique to ERP modpack, appearing nowhere else on the 112 Operator Workshop.<br /><img src="images/flags/112.png" alt="112" />&nbsp;&nbsp;Item is based on another asset from the 112 Operator Workshop.<br /><img src="images/flags/workshop.png" alt="workshop" />&nbsp;&nbsp;Item is translation only and requires another 112 Operator Workshop mod to function in-game.</p>
